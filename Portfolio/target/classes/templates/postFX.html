Второе приложение выполняет более серьезный функционал. В нем реализован процесс авторизации, регистрации и личный кабинет для редактирования данных пользователя.

Так же организованно добавление статей на сайт и их просмотр. Хранение информации выполняется в базе данных MySQL. Данное приложение может быть использовано для работы с статьями сайта не используя браузер, что дает возможность не реализовывать данный функционал на сайте. Третье приложение, оно же итоговое, приложение для хранения ссылок с коротким именем.

Данное приложение дает возможность хранить ссылке в базе данных MySQL. К данным ссылкам можно получить доступ через сайт написанный на Spring. Более подробно ознакомится с работой приложения можно в видео по ссылке. В процессе создания приложений были разобраны технологии организации построения приложений с GUI, взаимодействия с базой данных MySQL используя SQL запросы, а так же хеширование паролей с использованием md5.
