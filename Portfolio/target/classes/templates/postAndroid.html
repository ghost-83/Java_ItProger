Android

    За время обучения Android разработки, были изучены:
  • Работа с Activity, их жизненный цикл
  • Всплывающие подсказки, диалоговые окна и обработчик событий
  • Переход между страницами в приложении
  • Понятие фрагмента и работа с ними
  • Добавление звука в приложение
  • Работа с базой данных
  • Создание рабочего apk файла приложения
Так же было реализовано несколько приложений. Первое приложение выполняет функции конвертора масы(подобное приложение было выполнено на Java FX).

При вводе значения проводится конвертация. Принцип работы программы показан в видео по ссылке.
Cсылкa на git.

Второе приложение выполняет сохранение вводимых значений ы базе данных. В моем случае это доступный на мобильном телефоне список английских слов слов для повторения.

В приложении имеется возможность добавления новых слов с переводом, так и удаление изученных слов.

Все фразы сохраняются в базе данных, а в приложении отображены с помощью ListView. Более детально работу приложения можно увидеть при просмотре видео по ссылке или при установке приложения. Скачать.

Третье приложение(оно же итоговое), выполняет тот же функционал что и итоговые приложения на Java FX и Spring только без сохранения значений в базе данных.

Кстати в приложении имеется возможность перехода по адресам коротких ссылок в браузер. Можно создавать короткие ссылки с уникальным именем и нельзя создавать пустые ссылки. Подробней можно ознакомится в видео по ссылке.
