JAVA Spring

В процессе прохождения курса было реализовано несколько 2 проекта на Java Spring. Один из проектов - сайт с регистрацией и авторизацией. На сайте так же организованно разделение по ролям и возможность реактивировать данные (изменение почтового ящика, пароля и роли).

Имеется блог, организованна возможность добавления статей. У статей есть как предварительный просмотр так и подробный с отображением категории, даты создания и автора. Так же на сайте имеется небольшая панель администратора в которой можно посмотреть всех зарегистрированный пользователей и статьи написанные каждым из пользователей.

В процессе реализации данного проекта были использованы следующие модули:
1. Основной модуль для работы проекта с сервером TomcatWebServer
2. Библиотека spring-boot-starter-data-jpa для работы с MySQL и модуль с драйверами для работы SQL запросами mysql-connector-java
3. Для организации системы безопасности были использованы модули thymeleaf-extras-springsecurity5 и spring-boot-starter-security. С помощью модуля spring-boot-starter-security была организованна система авторизации, регистрации и ограничения доступа к материалом сайта. С помощью thymeleaf-extras-springsecurity5 был разделен доступ к страницам сайта согласно ролей пользователей.
4. В процессе разработки для обработки HTML кода было использовано 2 шаблонизатора mustache и thymeleaf. В итоговом варианте используется
шаблонизатор thymeleaf. Так же была использована техноология bootstrap 4 для построения адаптивных страниц сайта. Более детально модно увидеть в видео по ссылке.

Второй проект (он же итоговый), это небольшой сайт с возможностью сохранения ссылок в коротком виде.

В проекте организованна проверка на уникальности название короткой ссылки. Все сохраненные ссылки хранятся в базе данных MySQL. Так же организован доступ к данной таблице в базе через приложение написанное на JavaFX с подобным функционалом. Более детально модно увидеть в видео по ссылке.
